# Sum-Spiral-Edges
To sum the edges of a spiral 

### Input: 
2

### Expected Output

|1 | 2|
|--|---|
|4 |3|

**Edges: 1+2+3+4 = 10**

### Input: 
3

### Expected Output
|7|8|9|
|--|--|---|
|6 |1 | 2|
|5|4 |3|

**Edges: 7+9+3+5 = 24**

### Input: 
4

### Expected Output
|7|8|9|10
|--|--|---|--|
|6 |1 | 2| 11
|5|4 |3| 12
|16|15|14|13

**Edges: 7+10+13+16 = 46**
